中文

Anthropic Engineering 发了一篇架构稿,叫《Scaling Managed Agents:Decoupling the brain from the hands》。核心论点是:任何 harness 都默认了「Claude 现在做不了什么」,而这些假设会随模型能力迅速过期——Claude Sonnet 4.5 之前一接近 context limit 就急着收尾(他们管这叫「context anxiety」),而修法不是再训一版模型,是改 harness 怎么告诉模型「这事还在做」。

整套打法是把「认知」和「执行」剥开:model 这一层负责规划、推理、决策;每一条真要落到外面的动作——写文件、走网络、跑 shell——都走团队自己拥有、自己审计的 harness。权限、持久化、模型意图与外界的边界,都由 harness 兜底。能力越强,blast radius 越是被 harness 兜住,而不是模型本身兜住。

工程博客这一系列一直在讨论一件事:怎么搭 agent、怎么为长程任务设计 harness。这条线最深的共识是——harness 都在编码一种默认假设:「Claude 眼下做不了什么」。但这些假设会随模型能力持续进步而失效,必须不断拿出来重新审视。

举一个具体例子:之前我们发现 Claude Sonnet 4.5 在感知到快到 context 上限时会提前收尾——这种行为有时被叫做「context anxiety」。我们没有重训 Claude,而是改了 harness 把「这件事还在继续」的信号怎么传给模型。

English

Anthropic Engineering publishes a new architecture post: 'Scaling Managed Agents: Decoupling the brain from the hands.' The argument: harnesses encode assumptions about what Claude can't do on its own, and those assumptions go stale fast — Claude Sonnet 4.5 used to wrap up early when it sensed context anxiety, and the fix was never a model patch.

The pattern is to peel cognition from execution: the model plans and reasons, but every consequential action — file writes, network calls, shell commands — runs through a harness the team owns and audits. The harness owns permissions, persistence, and the boundary between Claude's intent and the outside world. As capabilities grow, the blast radius is capped by the harness, not by the model itself.

A running topic on the Engineering Blog is how to build effective agents and design harnesses for long-running work. A common thread across this work is that harnesses encode assumptions about what Claude can't do on its own. However, those assumptions need to be frequently questioned because they can go stale as models improve.

As just one example, in prior work we found that Claude Sonnet 4.5 would wrap up tasks prematurely as it sensed its context limit approaching—a behavior sometimes called 'context anxiety.' We addressed this by improving how the harness itself signals ongoing work to the model, without retraining Claude.
