中文

Anthropic 工程团队详解 Managed Agents:把 session、harness、sandbox 三件套都做成虚拟化抽象,让[大脑]——也就是 agent 调度循环——和[双手]——也就是实际执行环境——彻底解耦,这样单一接口可以撑过任何一种底层实现,不再养[宠物容器]。

Scaling Managed Agents: Decoupling the brain from the hands

A common thread across this work is that harnesses encode assumptions about what Claude can't do on its own. However, those assumptions need to be frequently questioned because they can go stale as models improve. As just one example, in prior work we found that Claude Sonnet 4.5 would wrap up tasks prematurely as it sensed its context limit approaching—a behavior sometimes called 'context anxiety.' We addressed this by adding context resets to the harness. But when we used the same harness on Claude Opus 4.5, we found that the behavior was gone. The resets had become dead weight.

English

Anthropic Engineering: a deep dive on Managed Agents — virtualized session / harness / sandbox abstractions that decouple the 'brain' (the agent loop) from the 'hands' (the execution environment), so a single interface can outlive any one implementation.
