Amjad Masad(Replit CEO)一条推把 inference-hardware 的故事讲完了:「AI is expensive to run partly because most workloads today run on generic hardware designed pre-LLMs. Etched is the first system designed from the ground up for modern inference」。

经济论断是结构性的:今天的 GPU 硅是通用目的,inference cost-of-goods 反映了这种通用性。一个为 transformer inference 专门设计的 chip(不要 general matrix multiply 兜底、不要 legacy 支持)有可能打到 5-10x perf-per-watt,这才是为什么 inference cost 的对话未来 18 个月会从「更便宜的模型」转向「更便宜的硅」的真实原因。Replit 角度:Replit 给每一个用户跑 inference,inference cost 是他们 first-order 的 P&L 行,Etched 这类系统会改变 agentic product development 的 unit economics。

Amasad 一条推把 inference hardware 的故事讲完:Etched 是为现代 inference 重新设计的首款系统,大部分 workload 跑在 pre-LLM 时代的通用硬件上是 AI 「贵」的真实原因;Replit 的角度是 inference cost 是 first-order P&L。

English

Amjad Masad (Replit CEO) makes the inference-hardware case in one tweet: 'AI is expensive to run partly because most workloads today run on generic hardware designed pre-LLMs. Etched is the first system designed from the ground up for modern inference.'

The economic claim is structural: GPU silicon today is general-purpose, and the inference cost-of-goods reflects that generality. A chip designed specifically for transformer inference (no general matrix multiply fallback, no legacy support) can plausibly hit 5-10x perf-per-watt, which is the real reason the cost-of-inference conversation is going to move from 'cheaper models' to 'cheaper silicon' over the next 18 months. The Replit angle: Replit runs inference for every user, so inference-cost is a first-order P&L line for them, and an Etched-class system would change the unit economics of agentic product development.
