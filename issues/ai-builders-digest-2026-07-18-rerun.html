Benedict Evans：先把 AGI 放到一边，看 foundation model 的 value 怎么落 / Benedict Evans: Set AGI Aside, Watch Where the Value Lands in Foundation Models

Unsupervised Learning 第 91 期请来 Evans。两张卡：value accrual 那条 enterprise software 拆解 + 「像不像 electricity」那条反向拆。

UL
Unsupervised Learning
Redpoint 出品的 AI 投资与技术播客
中文

Unsupervised Learning 第 91 期请来 Benedict Evans（Jacob Efron / Redpoint 主理）。Evans 整集的判断骨架：「如果我们不能定义 AGI 是什么，那就没法讨论它的影响——先把 AGI 放到一边，回到 enterprise software 这一层，否则聊的就是凌晨两点微醺哲学系研究生的自言自语。」

Evans 对 foundation model 估值的判断很具体：model provider 这一层目前「the one big market we've had so far in LLM world has actually proven to be very good for the model providers」——头部模型公司在 LLM 第一个真正成立的市场里跑出了完整的盈利结构。

Evans 的反向论证：value accrual 不会自动落在 model 这一层。Mobile / PC / cloud 都有过类似的判断（每一层都说自己最值钱），最后的事实是「集成到最底层 substrate」和「拆出来做成 vertical SaaS」的两端都能赚钱，但中间那层通用工具最容易蒸发。foundation model 是不是同一张剧本，现在定不下来。

我们这里其实已经进了 logical fallacy 的区域。Hinton 十年前说要停训放射科医生——问题是他根本不知道放射科医生实际在做什么。机器学习能自动化一个 narrow task，但 broad job 不等于做这个 narrow task。

LLM 至今跑出来的那一个真正能成立的市场，对 model provider 来说其实已经很好了。

如果回头看 enterprise software：半数 enterprise software 你去看它根本 get 不到，客户也 get 不到。我们每天用的工具里，一大半第一次见到都觉得「这有什么用」。
