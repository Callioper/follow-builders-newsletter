The Claude Loop Prompt / 让 Claude 跑长任务的节奏 prompt

TRQ 转发的这条 prompt 是今天全网最高赞的方法论分享——它解决了 Claude Code 长任务最容易翻车的问题。

TL
Tao Li
@trq212
中文

TRQ 问了 Anthropic 内部的同事：你们是怎么和 Claude 保持同步、并且完整理解它正在做的工作的？回答里他最喜欢的是 Suzanne 写的那段 prompt——一句话让 Claude 保持诚实、用短进度更新、在每一步把假设和风险说出来。

机制很有意思：不是试图监控整个任务，而是让 agent 用小段落把工作讲出来。用户拿到的是一段一段可以快速扫的 checkpoint，而不是任务结束后一整面墙的总结。

TRQ 转发 Suzanne 的 prompt，让 Claude 短更新、暴露假设和风险。

6183 赞。Suzanne 还配合 voice mode 用，让 agent 能自然打断她。

English

TRQ polled other Anthropic folks on how they stay in the loop with Claude and fully understand the work being done. The winning answer came from Suzanne: a single prompt that asks Claude to be honest, give short progress updates, and surface assumptions or risks at each step.

The mechanism is interesting: instead of trying to monitor the whole task, the agent narrates its work in small chunks. The user gets a steady stream of checkpoints they can actually scan, rather than a wall of text at the end.

6183 likes makes this the most resonant method-share of the day. Suzanne also mentioned she uses this together with voice mode so the agent can interrupt her naturally when it needs input.

Competitive Advantage in the Agent Era / Agent 时代的护城河

当每个对手都能用上同样的模型时，企业怎么建立差异？Levie 和 Garry Tan 从不同角度给出了答案。

AL
Aaron Levie
Box CEO
@levie
中文

Aaron Levie 抛出了这个时代的核心问题：当你的对手也能用上同样的 AI 模型和智能时，你怎么建立持续的优势？

他的答案是他最近几周一直在搭的那个结论：护城河不是模型本身，而是专有的工作流、客户关系、数据飞轮、集成的深度。Agent 会变成商品，能跑 Agent 的 context 不会。

对 builder 的含义是：赢的那家公司不是拥有最好 agent 的那家，而是把 agent 缝进真实客户真实运营里缝得最紧的那家。

Levie：Agent 时代护城河不是模型，是专有 workflow、数据飞轮和客户集成深度。

Agent 会商品化，context 不会。

English

Aaron Levie poses the defining question of the era: when your competitor has access to the same AI models and intelligence, how do you build a durable advantage?

His answer, which he has been building toward for weeks: the moat is not the model, it is the proprietary workflow, the customer relationship, the data flywheel, the integration depth. Agents are commoditized; the context in which they run is not.

The implication for builders: the company that wins is not the one with the best agent, it's the one that wires the agent most tightly into the real operations of a real customer.

As we enter the era of AI agents, one of the defining questions is how you develop competitive advantage when your competitor has access to the same AI models and intelligence as you.
