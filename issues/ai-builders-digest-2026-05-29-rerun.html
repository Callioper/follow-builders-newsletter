Claude Blog
Anthropic 官方博客
中文

Anthropic 的 Claude Managed Agents 推出三大更新：「做梦」作为研究预览、基于 outcomes 的评估、以及多 agent 编排和 webhooks。

「做梦」是最具概念趣味的：一个定期执行的过程，审查你的 agent 会话和记忆库，提取模式，整理记忆，让 agent 在没有直接人工干预的情况下随着时间改进。你选择有多少控制权——自动应用还是经过审查再落地。

Outcomes 功能让你写一个描述成功标准的评分标准，一个独立的 grader 在自己的 context 窗口里评估输出——所以评估不会受到 agent 推理的影响。Agent 迭代直到 grader 通过。

多 agent 编排和 webhooks 完善了这个图景：多个专精 agent 协调的复杂工作流，外部系统触发事件。这是 agentic 产品的基础设施，不只是实验。

今天推出「做梦」作为研究预览，还有 outcomes、多 agent 编排和 webhooks。做梦是定期审查会话和记忆、提取模式、让 agent 自我改进的机制。Outcomes 让你写成功标准，由独立 grader 评估。多 agent 编排支持复杂工作流。

English

Anthropic's Claude Managed Agents launches three major updates: 'dreaming' as a research preview, outcomes-based evaluation, and multiagent orchestration with webhooks.

Dreaming is the most conceptually interesting: a scheduled process that reviews your agent sessions and memory stores, extracts patterns, and curates memories so agents improve over time without direct human intervention. You choose how much control—automatic or reviewed before landing.

The outcomes feature lets you write a rubric describing what success looks like, and a separate grader evaluates the output in its own context window—so the evaluation isn't influenced by the agent's reasoning. Agents iterate until the grader passes.

Multiagent orchestration and webhooks complete the picture: complex workflows where multiple specialized agents coordinate, with external systems triggering events. This is infrastructure for agentic products, not just experiments.

Today we're launching dreaming in Claude Managed Agents as a research preview. Dreaming extends memory by reviewing past sessions to find patterns and help agents self-improve. We're also making outcomes, multiagent orchestration, and webhooks available to developers building with Managed Agents.
