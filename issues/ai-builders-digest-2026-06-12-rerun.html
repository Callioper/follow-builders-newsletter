Loopcraft 与 harness / Loopcraft and the Agent Harness

swyx 的 loopcraft 帖给出一句框架:整场游戏是把 loop 堆高,模型变强时把 loop 往上提一档吃杠杆。Logan Kilpatrick 在 Training Data 上把 Google 的 Antigravity 摆成 harness through-line,同一套驱动 Search / Gemini app / AI Studio / Cloud。Aaron Levie 拿 1,640 人 IT 调研补一枪,AI 采纳最深的公司招人计划最大,agentic AI 抬的是 onboard 门槛。

S
Swyx
AI 工程师
@swyx
中文

swyx 的 loopcraft 帖压缩成一句:「下个一百年整场游戏的玩法,就是把 loop 堆得尽可能高效。每个阶段的早期,知道什么时候该往下一档 loop 走(为可靠性)是值钱的;但更值钱的,是知道模型变强之后怎么把 loop 往上抬(为杠杆)。」

未来十年要付钱的两个技能:知道什么时候主动退一档(保稳)、知道什么时候主动提一档(吃杠杆)。「如果这两件事你想不明白,输了别酸。」这个框架是普适的——编码 agent、内容 loop、infra 自动化、RL 训练,甚至个人反馈 loop 都吃得下。

swyx loopcraft:整场游戏是把 loop 堆高;早期会往下退一档保稳,更值钱的是模型变强时把 loop 往上提一档吃杠杆。

English

swyx's loopcraft thread, condensed: 'One might argue the entire game of the next century is to be able to stack loops as effectively as possible. In the early days of each phase, it will be valuable to know when to go DOWN a loop when things go wrong (for reliability)… but it will probably be more valuable to know how to go UP a loop as models improve (for leverage).'

The two skills the next decade will pay for: knowing when to retreat one loop down (reliability) and when to promote one loop up (leverage). 'If you don't figure out how to do this, don't be salty when you lose to those that do.' The framing is universal — applies to coding agents, content loops, infra automation, RL training, even a personal feedback loop.

## On Loopcraft One might argue the entire game of the next century is to be able to stack loops as effectively as possible. In the early days of each phase, it will be valuable to know when to go **DOWN** a loop when things go wrong (for reliability)… but it will probably be more valuable to know how to go **UP** a loop as models improve (for leverage). If you don't figure out how to do this, don't be salty when you lose to those that do.
