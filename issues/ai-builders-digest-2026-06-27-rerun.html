中文

Noam Brown(OpenAI 研究员,Libratus / Pluribus / Cicero Diplomacy agent 的共同作者)上 No Priors 和 Sarah Guo 一起讲:行业评测 frontier model 的方式从根本上就是 broken。每次发 model 都附带的「single number on a single benchmark」那张 grid,没有 control for test-time compute,所以一个在纸上看起来「更差」的 model,在实际使用上可能「好得多」——因为它用同样的 think budget 做到了更多。

English

Noam Brown (OpenAI research scientist, co-author of Libratus / Pluribus / the Cicero Diplomacy agent) joins No Priors with Sarah Guo to argue that the way the industry evaluates frontier models is fundamentally broken. The single-number-on-a-single-benchmark grid that ships with every release does not control for test-time compute, so a model that is 'worse' on paper can be 'much better' in actual use because it gets more done with the same think budget.
